Guia Interativo de Instalação FVM

Siga estes passos para instalar e configurar o Flutter Version Manager (FVM) no seu ambiente Windows de forma simples e eficiente.

1

Instalação do Chocolatey

O Chocolatey é um gerenciador de pacotes para Windows que simplifica a instalação de softwares. Primeiro, precisamos instalá-lo.

  1. Abra o **PowerShell** como **Administrador**. Você pode fazer isso pesquisando "PowerShell" no menu Iniciar, clicando com o botão direito e selecionando "Executar como administrador".
  2. Copie e execute o comando abaixo para instalar o Chocolatey.
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

Após a conclusão, verifique a instalação com:

choco --version
2

Instalação do FVM e Dart SDK

Com o Chocolatey pronto, podemos instalar o Dart SDK (uma dependência do FVM) e o próprio FVM com dois simples comandos.

1. Instale o Dart SDK:

choco install dart-sdk

2. Instale o FVM:

choco install fvm

Quando o Chocolatey perguntar se você quer executar o script, digite `A` para `All scripts` e pressione Enter para permitir a instalação.

3

Configuração Global do FVM

Agora, vamos usar o FVM para baixar uma versão do Flutter e defini-la como padrão em seu sistema.

1. Instale a versão `stable` do Flutter:

fvm install stable

2. Defina-a como a versão global:

fvm global stable

Após o último comando, o FVM mostrará um aviso com o caminho que você precisa adicionar ao seu PATH. Será algo como: C:\Users\<seu-usuario>\fvm\default\bin.

4

Adicionando ao PATH do Windows

Este é o passo mais importante. Ele permite que você use o comando `flutter` diretamente no terminal, sem precisar digitar `fvm flutter`.

  1. Pesquise por **"variáveis de ambiente"** no menu Iniciar e selecione **"Editar as variáveis de ambiente do sistema"**.
  2. Na janela que abrir, clique em **"Variáveis de Ambiente..."**.
  3. Na seção superior ("Variáveis de usuário"), selecione a variável **"Path"** e clique em **"Editar..."**.
  4. Clique em **"Novo"** e cole o caminho que o FVM forneceu no passo anterior.
C:\Users\<seu-usuario>\fvm\default\bin

5. Clique em **"OK"** em todas as janelas para salvar. **Lembre-se de fechar e abrir um novo terminal** para que as alterações tenham efeito.

5

Gerenciando Versões do Flutter

Agora você está pronto para gerenciar múltiplas versões do Flutter. Veja como alternar entre a versão global e versões específicas por projeto.

A versão global é usada por padrão quando você não está dentro de um projeto que tem uma versão específica definida.

Use `fvm global ` para alterar:

fvm global 3.3.0
fvm global beta

Essa é a principal vantagem do FVM. Cada projeto pode ter sua própria versão do Flutter, evitando conflitos.

  1. Navegue até o diretório do seu projeto no terminal.
  2. Instale a versão desejada para aquele projeto:
fvm install 3.0.0

3. Defina a versão para o projeto. O FVM criará uma pasta `.fvm` para rastreá-la.

fvm use 3.0.0

Verificação

Para verificar qual versão do Flutter está ativa (seja global ou do projeto), use:

flutter --version
